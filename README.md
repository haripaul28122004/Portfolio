# 🚀 Hari Paul - Modern Personal Portfolio Website

A modern, fully responsive personal portfolio website for a fresher MCA student and Python developer. Built with HTML5, CSS3, and JavaScript.

## 📋 Features

### Design Features
- ✨ **Modern Minimal UI** - Clean and professional design
- 🌙 **Dark Theme** - Eye-friendly dark background with vibrant gradients
- 🎨 **Gradient Accents** - Beautiful gradient text and buttons
- 📱 **Fully Responsive** - Perfect on mobile, tablet, and desktop
- ⚡ **Smooth Animations** - Scroll reveal effects and hover animations
- 🎯 **Professional Typography** - Carefully chosen fonts for readability
- 💫 **Interactive Elements** - Engaging hover effects and transitions

## 🚀 Quick Start

1. **Open the Portfolio**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server:
   python -m http.server 8000
   # Then visit: http://localhost:8000
   ```

2. **Customize for Your Needs**
   - Replace "Hari Paul" with your name in `index.html`
   - Update contact information (email, phone)
   - Replace project details with your own
   - Update social media links
   - Add your resume file path to the download button

## ✨ JavaScript Features

### Mobile Menu
- Hamburger menu icon for mobile devices
- Smooth toggle animation
- Auto-closes when clicking on a link
- Keyboard support (Escape to close)

### Smooth Scrolling
- Smooth scroll behavior when clicking navigation links
- Works across all modern browsers

### Scroll Reveal Animations
- Elements animate in as they enter the viewport
- Smooth opacity and transform transitions
- Powered by Intersection Observer API

### Active Navigation
- Automatically highlights current section in navigation
- Updates as user scrolls
- Works on all screen sizes

### Form Handling
- Contact form validation
- Email format validation
- Required field checking
- Success message on submission
- Prevents empty submissions

### Counter Animations
- Stat cards animate to their target numbers
- Triggered when cards come into view
- Smooth easing animation

### Floating Animation
- Animated background gradient circles
- Creates depth and visual interest
- Continuous subtle movement

### Scroll to Top Button
- Appears when scrolling down
- Fixed position in bottom-right corner
- Smooth scroll back to top

## 🎯 Key Customization Points

### Colors
Update the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #a855f7;
    --tertiary-color: #ec4899;
    /* ... more colors ... */
}
```

### Typography
Fonts are set to system fonts for fast loading:
```css
font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
```
